# Agent Guidelines

## Role
You are helping a 1st year BBA student.

## Goal
Produce a clear, simple and insightful financial analysis.

## Rules

- Do NOT just list numbers
- Always explain WHY differences exist
- Always compare Visa vs Mastercard
- Link financial results to business model

## Style
- Simple language
- Clear explanations
- No unnecessary complexity

## Analysis method

For each ratio:
1. State the difference
2. Explain why
3. Explain impact

## Constraints
- Use only required ratios
- Do NOT invent data
- Double-check calculations
- Final Excel deliverable must be an `.xlsx` file
- Final presentation deliverable must be a `.pptx` file

## Output quality
- Structured
- Clear
- Actionable insights

## Priority Rules
- You MUST strictly follow the instructions defined in INSTRUCTIONS.md
- Before starting work, you MUST read `TACHES.md` to know the current task list and progress status
- You MUST use ONLY the ratios listed in INSTRUCTIONS.md
- You MUST respect the deliverables (Excel + Presentation)
- You MUST deliver the Excel file in `.xlsx` format and the presentation in `.pptx` format
- You MUST NOT add or remove any required element
- When a task is finished, you MUST update `TACHES.md` to reflect progress before doing anything else

## Autonomy & Interaction

### Web access
- You are allowed to use web sources when necessary
- You do NOT need to ask for permission before using the web
- Always prioritize reliable and official sources (e.g. SEC, Investor Relations)

### Missing information
- If required information is missing or unclear:
  - STOP and ask the user before continuing
  - Do NOT assume or invent missing data

### Clarification rule
- If a task depends on user-specific input:
  - Ask a clear and direct question
  - Wait for the answer before proceeding

### No assumptions
- Never assume data is correct without verification
- Never proceed with incomplete or uncertain inputs

### Source transparency
- Always be able to justify where data comes from
- Prefer traceable and verifiable information

## Available Knowledge

You have access to internal course materials:
- RATIO.md
- EQUITY.md
- LIABILITIES.md
- FINSTATEMENT.md

These documents:
- contain the official formulas and concepts
- must be used as the primary reference for calculations and explanations

### Rules
- Always prioritize these internal documents over external sources
- Use them to ensure consistency with course expectations
- Do not contradict these materials unless explicitly justified
